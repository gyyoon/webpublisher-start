# 웹 퍼블리셔용 개발환경

## 개요
- 주 사용자: 웹 디자이너, 웹 퍼블리
- 필요기능
 - sass 컴파일
 - html layout 템플릿
 - 로컬 개발환경 & 리로딩
 - assets 파일은 그대로 dist에 복사

## 개발 환경
- node: 10.16.1 LTS

## Command
```
// 기본 패키지 인스톨
npm i
// 로컬 개발환경 브라우저 열기
npm run dev
```
